# P7-Groupomania

Voici la page du Projet 7 de la formation Développeur Web d'OpenClassRooms intitulé "Créer un réseau social d'entreprise"

Le projet consiste à construire un réseau social interne pour les employés de Groupomania. Le but de cet outil est de faciliter les interactions entre collègues. Le département RH de Groupomania a laissé libre cours à son imagination pour les fonctionnalités du réseau et a imaginé plusieurs briques pour favoriser les échanges entre collègues.

-----

Mise en route du projet

Pour le backend 

Framework untilisé : Express

pour cela dans votre Terminal placez vous dans le dossier backend 
et lancez nodemon server

si vous ne l'avez pas installé utilisez la commande qui suit

npm install -g nodemon

---

Pour le Frontend

Framework utilisé : ReactJS

dans votre Terminal placez vous dans le dossier frontend

et lancez yarn pour démarrer le Frontend en tapant la commande suivante

yarn start

si vous ne l'avez pas installé utilisez la commande qui suit 

npm install --global yarn

puis lancez yarn
